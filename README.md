Professor: Prof. Dr. Cristiano Both

## Descrição

Este projeto é um simulador de memória virtual implementado em Python. Ele simula o processo de gerenciamento de memória virtual, incluindo a alocação de páginas, substituição de páginas e manipulação de falhas de página.
